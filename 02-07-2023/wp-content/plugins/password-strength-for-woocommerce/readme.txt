Disables password strength enforcement in WooCommerce.

== Description ==

This plugin disables password strength enforcement in WooCommerce. Future versions may allow the user to change the minimum password strength rather than disabling it altogether.

Disabling the minimum password strength is not recommended from a security standpoint. Use at your own risk!


== Installation ==

1. Click "Plugins" > "Add New" in the WordPress admin menu.
1. Search for "Password Strength for WooCommerce".
1. Click "Install Now".
1. Click "Activate Plugin".

Alternatively, you can manually upload the plugin to your wp-content/plugins directory.
